准备工作

技术要求

  • React

由于Goji基于最新版本的React。 你应该先学习如何开发React应用。

  • React 钩子

我们鼓励你使用带钩子的函数式组件而不是类组件。 一些像 生命周期 这样的Goji功能只支持钩子 API。

  • 至少一个小程序平台的开发经验

我们建议学习微信小程序。 其他平台也是可以的， 包括 百度, 支付宝, QQ 和 头条。

代码编辑器

我们强烈建议使用 VSCode 作为你的代码编辑器以获取更好的开发者体验。 我们也推荐您安装并启用这些VSCode 插件。

  • ESLint
  • Prettier
  • StyleLint
